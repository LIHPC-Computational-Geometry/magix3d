Outils
######

.. _bibop3d:

Bibop3D
*******

Bibop3D est une bibliothèque permettant la réalisation de maillage d'objets paramétrés tridimensionnels.

.. _revolMdlMesh:

revolMdlMesh
************

revolMdlMesh est un script Python qui utilise des fonctionnalités de :ref:`bibop3d`. Ce script permet de construire un maillage 3D en partant du modèle 2D (enrichi d'un support :ref:`o-grid`).
Cette fonctionnalité de création du maillage par révolution d'un modèle 2D est également disponible dans :ref:`Magix3D<magix3d>`.
